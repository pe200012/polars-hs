# Arrow C Data Interface Export Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** Add scoped `withArrowRecordBatch` export for `DataFrame -> ArrowSchema + ArrowArray` RecordBatch interop.

**Architecture:** Rust builds a top-level Arrow `StructArray` from a Polars `DataFrame`, exports it with `polars_arrow::ffi`, and stores the C structs in an opaque `phs_arrow_record_batch` handle. Haskell exposes a scoped callback API that keeps the handle alive while the callback uses the pointers, then finalizes the Rust handle.

**Tech Stack:** Haskell Stack/GHC 9.12.2, Hspec, Rust 2024, Polars 0.53.0, `polars-arrow` 0.53.0, cbindgen, Apache Arrow C Data Interface.

pub struct phs_arrow_record_batch {
    _private: [u8; 0],
}

struct ArrowRecordBatchHandle {
    schema: ArrowSchema,
    array: ArrowArray,
}

fn arrow_record_batch_into_raw(handle: ArrowRecordBatchHandle) -> *mut phs_arrow_record_batch {
    Box::into_raw(Box::new(handle)) as *mut phs_arrow_record_batch
}

unsafe fn arrow_record_batch_ref<'a>(ptr: *const phs_arrow_record_batch) -> PhsResult<&'a ArrowRecordBatchHandle> {
    if ptr.is_null() {
        Err(PhsError::invalid_argument("arrow record batch pointer was null"))
    } else {
        Ok(unsafe { &*(ptr as *const ArrowRecordBatchHandle) })
    }
}

unsafe fn arrow_record_batch_mut<'a>(ptr: *mut phs_arrow_record_batch) -> PhsResult<&'a mut ArrowRecordBatchHandle> {
    if ptr.is_null() {
        Err(PhsError::invalid_argument("arrow record batch pointer was null"))
    } else {
        Ok(unsafe { &mut *(ptr as *mut ArrowRecordBatchHandle) })
    }
}

- [x] **Step 3: Add export/free/accessor ABI functions**

Add:

```rust
#[unsafe(no_mangle)]
pub unsafe extern "C" fn phs_dataframe_to_arrow_record_batch(
    dataframe: *const phs_dataframe,
    out: *mut *mut phs_arrow_record_batch,
    err: *mut *mut phs_error,
) -> c_int {
    ffi_boundary(err, || {
        let out = unsafe { required_mut(out, "out") }?;
        *out = ptr::null_mut();
        let handle = unsafe { dataframe_ref(dataframe) }?;
        let df = handle.value.clone();
        let compat = CompatLevel::newest();
        let fields = df
            .columns()
            .iter()
            .map(|column| column.field().to_arrow(compat))
            .collect::<Vec<_>>();
        let arrays = df.rechunk_to_arrow(compat);
        let dtype = ArrowDataType::Struct(fields.clone());
        let struct_array = StructArray::new(dtype.clone(), df.height(), arrays, None);
        let schema_field = ArrowField::new("".into(), dtype, false);
        let schema = export_field_to_c(&schema_field);
        let array = export_array_to_c(Box::new(struct_array));
        *out = arrow_record_batch_into_raw(ArrowRecordBatchHandle { schema, array });
        Ok(())
    })
}

#[unsafe(no_mangle)]
pub unsafe extern "C" fn phs_arrow_record_batch_schema(batch: *mut phs_arrow_record_batch) -> *mut c_void {
    unsafe { arrow_record_batch_mut(batch) }
        .map(|handle| (&mut handle.schema as *mut ArrowSchema).cast::<c_void>())
        .unwrap_or(ptr::null_mut())
}

#[unsafe(no_mangle)]
pub unsafe extern "C" fn phs_arrow_record_batch_array(batch: *mut phs_arrow_record_batch) -> *mut c_void {
    unsafe { arrow_record_batch_mut(batch) }
        .map(|handle| (&mut handle.array as *mut ArrowArray).cast::<c_void>())
        .unwrap_or(ptr::null_mut())
}

#[unsafe(no_mangle)]
pub unsafe extern "C" fn phs_arrow_record_batch_free(batch: *mut phs_arrow_record_batch) {
    if batch.is_null() {
        return;
    }
    unsafe { drop(Box::from_raw(batch as *mut ArrowRecordBatchHandle)) };
}
```

```haskell
newtype ExportedArrowRecordBatch = ExportedArrowRecordBatch (ForeignPtr RawArrowRecordBatch)

withArrowRecordBatch :: DataFrame -> (Ptr schema -> Ptr array -> IO a) -> IO (Either PolarsError a)
withArrowRecordBatch df action =
    withDataFrame df $ \dfPtr ->
        arrowRecordBatchOut (phs_dataframe_to_arrow_record_batch dfPtr) $ \batch ->
            withExportedArrowRecordBatch batch action

arrowRecordBatchOut :: (Ptr (Ptr RawArrowRecordBatch) -> Ptr (Ptr RawError) -> IO CInt) -> (ExportedArrowRecordBatch -> IO a) -> IO (Either PolarsError a)
arrowRecordBatchOut create action =
    alloca $ \outPtr ->
        alloca $ \errPtr -> do
            poke outPtr nullPtr
            poke errPtr nullPtr
            status <- create outPtr errPtr
            if status == 0
                then do
                    ptr <- peek outPtr
                    if ptr == nullPtr
                        then pure (Left (nullPointerError "arrow record batch output"))
                        else bracket
                            (ExportedArrowRecordBatch <$> newForeignPtr phs_arrow_record_batch_free_funptr ptr)
                            (\(ExportedArrowRecordBatch foreignPtr) -> finalizeForeignPtr foreignPtr)
                            (\batch -> Right <$> action batch)
                else Left <$> (consumeError status =<< peek errPtr)

withExportedArrowRecordBatch :: ExportedArrowRecordBatch -> (Ptr schema -> Ptr array -> IO a) -> IO a
withExportedArrowRecordBatch (ExportedArrowRecordBatch foreignPtr) action =
    withForeignPtr foreignPtr $ \batchPtr -> do
        schema <- phs_arrow_record_batch_schema batchPtr
        array <- phs_arrow_record_batch_array batchPtr
        action (castPtr schema) (castPtr array)
```
